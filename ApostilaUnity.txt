Paradigmas da Unity:
-> Usa-se a linguagem de programação C#
-> Cenas
	Na Unity, o jogo é feito de cenas. Cada cena é uma fase, ou um menu.
-> Game Objects
	As cenas contém vários Game Objects.
-> Componentes
	Cada Game Object contém um ou vários componentes.

Sistema de física
-> Explicação sobre vetores da física e matemática reais.
-> Componente Rigidbody2D
-> Exemplo de física sendo usada no código:
	GetComponent<Rigidbody2D>().velocity = new Vector3(0,5,0);
	// a linha acima faz o objeto receber uma velocidade de 5, para cima.

Input
-> Exemplo no código.
Input.GetKeyDown(KeyCode.UpArrow);
// a linha acima retorna True caso a setinha para cima tenha sido apertada.
